# 📰 AI News Credibility Detector

## 📌 Project Overview

AI News Credibility Detector is a Machine Learning web application that predicts whether a news article is likely to be **Real** or **Fake** using Natural Language Processing (NLP) and Logistic Regression.

---

## 🚀 Features

- Predicts Real or Fake News
- Confidence Score
- Machine Learning Model
- Flask Backend
- HTML & CSS Frontend
- User-Friendly Interface

---

## 🛠️ Technologies Used

- Python
- Flask
- HTML
- CSS
- Pandas
- Scikit-learn
- Joblib
- TF-IDF Vectorizer
- Logistic Regression
